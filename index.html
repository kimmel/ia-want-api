Internet Archive: Do we want this book?

This application programming interface (API) is a single endpoint which returns the Internet Archive's status on whether or not they need a book for their collection. Books are identified by their ISBN.

Request Format

GET http://want.archive.org/api?isbn=id

id is an ISBN10 or ISBN13 value with or without dashes.

Optional Parameters

Add &human to the end of your request for a human readable response.

Add &image to the end of your request to return the cover of the book if available or an image response. Note: &image overrides &human if they are both used in a query string.

GET http://want.archive.org/api?isbn=id&human

GET http://want.archive.org/api?isbn=id&image

JSON response definition:

status

value description
fail We failed to process the request. The submitted ISBN was invalid.
success The request was serviced successfully.

result

value description
-1 We failed to process the request. The submitted ISBN was invalid.
0 The request was serviced successfully, but we have two copies, and do not want more.
1 The request was serviced successfully, and we have no copies. We want it.
2 The request was serviced successfully, and we have 1 copy already. We want it.

description

A "human-parsable" description field

failure
do_not_want
want_for_ia_pa [needs example]
want_for_alt_pa

identifier

String when result = 2. This is the identifier already assigned on the cluster for that ISBN

value description
-1 No identifier designated for this ISBN.
string Designated identifier for this ISBN.

Examples:

Successful requests

curl http://want.archive.org/api?isbn=1930841426

{

    "status": "success",
    "result": "0",
    "description": "do_not_want",
    "identifier": "mozartsmagicflut00burt"

}

&human response - We already have two copies, and don't need anymore. Thanks anyway!

curl http://want.archive.org/api?isbn=9780452295971

{

    "status": "success",
    "result": "2",
    "description": "want_for_alt_pa",
    "identifier": "esquarednovel00matt"

}

curl http://want.archive.org/api?isbn=9780452295971&human response - We've scanned this book, and have a copy, but we need another. Please send it to us!

Failed request

curl http://want.archive.org/api?isbn=9781416913849f

{

    "status": "fail",
    "result": "-1",
    "description": "failure",
    "identifier": "-1"

}

curl http://want.archive.org/api?isbn=9781416913849f&human - We couldn't process the request. Please feed us valid ISBNs!
